Willkommen zum HTML-Editor Workshop Bienvenue à l'atelier HTML-Editor

In diesem Workshop lernen Sie, mit Hilfe von KI-Bots interaktive HTML-Lernseiten zu erstellen — ohne Programmierkenntnisse! Dans cet atelier, vous apprendrez à créer des pages d'apprentissage HTML interactives à l'aide de bots IA — sans connaissances en programmation !

🎯 Ziel des Workshops 🎯 Objectif de l'atelier

Sie werden am Ende dieses Workshops in der Lage sein, eigenständig interaktive HTML-Seiten zu erstellen, die Sie als Lernmaterialien im Unterricht einsetzen können. Dazu nutzen Sie einen KI-Bot (z. B. ChatGPT, Claude, Copilot), dem Sie über sogenannte Prompts Anweisungen geben. À la fin de cet atelier, vous serez capable de créer de manière autonome des pages HTML interactives que vous pourrez utiliser comme supports pédagogiques en classe. Pour cela, vous utiliserez un bot IA (p. ex. ChatGPT, Claude, Copilot) auquel vous donnerez des instructions via des prompts.

✅ Was Sie erreichen werden ✅ Ce que vous allez accomplir

  • Verstehen, wie der Prozess von Prompt zu HTML-Seite funktioniert Comprendre comment fonctionne le processus du prompt à la page HTML
  • Eigene interaktive Lernseiten erstellen (Quizze, Übungen, Infoseiten) Créer vos propres pages d'apprentissage interactives (quiz, exercices, pages d'information)
  • Prompts iterativ verbessern für bessere Ergebnisse Améliorer les prompts de manière itérative pour de meilleurs résultats
  • Die fertigen Seiten an Lernende verteilen Distribuer les pages finies aux apprenants

💡 Keine Vorkenntnisse nötig 💡 Aucune connaissance préalable requise

Sie brauchen keine Programmierkenntnisse. Der KI-Bot schreibt den Code für Sie. Sie konzentrieren sich darauf, was die Seite können soll — der Bot kümmert sich um das Wie. Vous n'avez pas besoin de connaissances en programmation. Le bot IA écrit le code pour vous. Vous vous concentrez sur ce que la page doit faire — le bot s'occupe du comment.

📋 Schritt-für-Schritt-Anleitung 📋 Instructions étape par étape

  1. KI-Bot öffnen: Öffnen Sie einen KI-Bot Ihrer Wahl im Browser — z. B. ChatGPT (chat.openai.com), Claude (claude.ai), oder Copilot (copilot.microsoft.com). Ouvrir le bot IA : Ouvrez un bot IA de votre choix dans le navigateur — p. ex. ChatGPT (chat.openai.com), Claude (claude.ai) ou Copilot (copilot.microsoft.com).
  2. Prompt eingeben: Schreiben Sie Ihren Prompt (Anweisung) in das Chatfeld. Beschreiben Sie, was Ihre HTML-Seite enthalten und können soll (siehe Tab «Beispiel-Prompts»). Entrer le prompt : Écrivez votre prompt (instruction) dans le champ de chat. Décrivez ce que votre page HTML doit contenir et faire (voir l'onglet « Exemples de prompts »).
  3. Ergebnis kopieren: Der Bot gibt Ihnen HTML-Code zurück. Kopieren Sie den gesamten Code (alles von <!DOCTYPE html> bis </html>). Copier le résultat : Le bot vous renvoie du code HTML. Copiez tout le code (tout de <!DOCTYPE html> à </html>).
  4. In Textdatei einfügen: Öffnen Sie einen Texteditor (z. B. Editor/Notepad auf Windows, TextEdit auf Mac) und fügen Sie den kopierten Code ein. Coller dans un fichier texte : Ouvrez un éditeur de texte (p. ex. Bloc-notes sur Windows, TextEdit sur Mac) et collez le code copié.
  5. Als HTML-Datei speichern: Speichern Sie die Datei mit der Endung .html (siehe technische Anleitung unten). Enregistrer en tant que fichier HTML : Enregistrez le fichier avec l'extension .html (voir les instructions techniques ci-dessous).
  6. Im Browser öffnen: Doppelklicken Sie auf die gespeicherte .html-Datei — sie öffnet sich in Ihrem Browser! Ouvrir dans le navigateur : Double-cliquez sur le fichier .html enregistré — il s'ouvre dans votre navigateur !

📝 Beispiel-Prompts für interaktive Lernseiten 📝 Exemples de prompts pour des pages d'apprentissage interactives

Hier finden Sie Beispiel-Prompts, die Sie direkt in einen KI-Bot kopieren können. Passen Sie den Inhalt an Ihre Bedürfnisse an! Voici des exemples de prompts que vous pouvez copier directement dans un bot IA. Adaptez le contenu à vos besoins !

🧩 Quiz-Seite 🧩 Page de quiz

Prompt:

Erstelle eine vollständige HTML-Seite mit einem interaktiven Quiz zum Thema [Ihr Thema]. Das Quiz soll 5 Multiple-Choice-Fragen enthalten. Jede Frage hat 4 Antwortmöglichkeiten. Wenn man eine Antwort anklickt, wird sofort angezeigt, ob sie richtig oder falsch ist (grün/rot). Am Ende wird der Punktestand angezeigt. Verwende ein modernes, ansprechendes Design mit Farben. Die Seite soll vollständig selbständig funktionieren (alles in einer HTML-Datei, inklusive CSS und JavaScript). Crée une page HTML complète avec un quiz interactif sur le thème [votre thème]. Le quiz doit contenir 5 questions à choix multiples. Chaque question a 4 options de réponse. Quand on clique sur une réponse, on voit immédiatement si elle est correcte ou incorrecte (vert/rouge). À la fin, le score total est affiché. Utilise un design moderne et attrayant avec des couleurs. La page doit fonctionner de manière autonome (tout dans un seul fichier HTML, y compris CSS et JavaScript).

📚 Lernkarten (Flashcards) 📚 Cartes d'apprentissage (Flashcards)

Prompt:

Erstelle eine HTML-Seite mit interaktiven Lernkarten (Flashcards) zum Thema [Ihr Thema]. Die Karten sollen sich beim Klick umdrehen (Vorderseite: Frage/Begriff, Rückseite: Antwort/Definition). Es sollen 10 Karten vorhanden sein. Füge Navigationsbuttons hinzu (Vorherige/Nächste). Die Seite soll ein freundliches, farbiges Design haben. Alles in einer einzigen HTML-Datei. Crée une page HTML avec des cartes d'apprentissage interactives (flashcards) sur le thème [votre thème]. Les cartes doivent se retourner au clic (recto : question/terme, verso : réponse/définition). Il doit y avoir 10 cartes. Ajoute des boutons de navigation (Précédent/Suivant). La page doit avoir un design amical et coloré. Le tout dans un seul fichier HTML.

🎮 Zuordnungsübung (Drag & Drop) 🎮 Exercice d'association (Drag & Drop)

Prompt:

Erstelle eine HTML-Seite mit einer Drag-and-Drop-Zuordnungsübung zum Thema [Ihr Thema]. Auf der linken Seite stehen Begriffe, auf der rechten Seite Definitionen. Die Lernenden sollen die Begriffe per Drag & Drop den richtigen Definitionen zuordnen. Bei richtiger Zuordnung wird das Paar grün markiert. Am Ende gibt es einen «Überprüfen»-Button. Modernes, einladendes Design. Alles in einer HTML-Datei. Crée une page HTML avec un exercice d'association en drag-and-drop sur le thème [votre thème]. À gauche se trouvent les termes, à droite les définitions. Les apprenants doivent faire glisser les termes vers les bonnes définitions. En cas d'association correcte, la paire est marquée en vert. À la fin, il y a un bouton « Vérifier ». Design moderne et accueillant. Le tout dans un seul fichier HTML.

📖 Interaktive Infoseite mit Tabs 📖 Page d'information interactive avec onglets

Prompt:

Erstelle eine HTML-Seite, die als interaktive Infoseite zum Thema [Ihr Thema] dient. Die Seite soll verschiedene Abschnitte in Tabs organisieren. Jeder Tab behandelt ein Unterthema. Verwende aufklappbare Bereiche (Akkordeons) für zusätzliche Details. Füge einen Fortschrittsbalken hinzu, der anzeigt, wie viele Abschnitte schon gelesen wurden. Farbiges, modernes Design. Alles in einer HTML-Datei. Crée une page HTML servant de page d'information interactive sur le thème [votre thème]. La page doit organiser différentes sections en onglets. Chaque onglet traite un sous-thème. Utilise des zones dépliables (accordéons) pour les détails supplémentaires. Ajoute une barre de progression qui montre combien de sections ont été lues. Design coloré et moderne. Le tout dans un seul fichier HTML.

🚀 Eigene Prompts erstellen 🚀 Créer ses propres prompts

Jetzt sind Sie dran! Erstellen Sie Ihre eigenen Prompts für interaktive Lernseiten. Der Schlüssel zum Erfolg: iteratives Vorgehen. C'est à votre tour ! Créez vos propres prompts pour des pages d'apprentissage interactives. La clé du succès : une approche itérative.

🔄 Was bedeutet «iterativ»? 🔄 Que signifie « itératif » ?

Iterativ bedeutet, dass Sie Ihren Prompt schrittweise verbessern. Sie schreiben einen ersten Entwurf, schauen sich das Ergebnis an und passen den Prompt dann an. Dieser Kreislauf wiederholt sich, bis das Ergebnis Ihren Vorstellungen entspricht. Itératif signifie que vous améliorez votre prompt étape par étape. Vous écrivez un premier brouillon, examinez le résultat, puis adaptez le prompt. Ce cycle se répète jusqu'à ce que le résultat corresponde à vos attentes.

1

Prompt schreiben Écrire le prompt

2

Ergebnis testen Tester le résultat

3

Prompt anpassen Adapter le prompt

Wiederholen Répéter

💡 Tipp: Beginnen Sie einfach und fügen Sie bei jeder Iteration Details hinzu. So vermeiden Sie übermässig komplexe Prompts. 💡 Conseil : Commencez simplement et ajoutez des détails à chaque itération. Cela évite les prompts trop complexes.

✅ Was einen guten Prompt ausmacht ✅ Ce qui fait un bon prompt

Um eine gute HTML-Seite vom KI-Bot zu erhalten, sollte Ihr Prompt diese Punkte abdecken: Pour obtenir une bonne page HTML du bot IA, votre prompt devrait couvrir ces points :

  • Zweck: Was ist das Ziel der Seite? (z. B. Lernstoff üben, Wissen überprüfen, informieren) Objectif : Quel est le but de la page ? (p. ex. pratiquer la matière, vérifier les connaissances, informer)
  • Zielgruppe: Wer nutzt die Seite? (z. B. Lernende EBA, Berufslernende, Erwachsene) Public cible : Qui utilise la page ? (p. ex. apprenants AFP, apprentis, adultes)
  • Nutzung: Wie wird das Ergebnis eingesetzt? (z. B. im Unterricht, als Hausaufgabe, zur Prüfungsvorbereitung) Utilisation : Comment le résultat sera-t-il utilisé ? (p. ex. en cours, comme devoir, pour la préparation aux examens)
  • Interaktivität: Welche Art von Interaktion? (z. B. Quiz, Drag & Drop, Lückentext, Aufklappelemente) Interactivité : Quel type d'interaction ? (p. ex. quiz, drag & drop, texte à trous, éléments dépliables)
  • Inhalt: Welchen Inhalt soll die Seite haben? Beschreiben Sie ihn möglichst genau oder liefern Sie ihn direkt mit. Contenu : Quel contenu la page doit-elle avoir ? Décrivez-le aussi précisément que possible ou fournissez-le directement.
  • Design: Wie soll die Seite aussehen? (z. B. farbenfroh, modern, schlicht) Design : À quoi doit ressembler la page ? (p. ex. colorée, moderne, simple)

🔒 Datenschutz bei der KI-Nutzung 🔒 Protection des données lors de l'utilisation de l'IA

Beim Arbeiten mit KI-Bots ist der Datenschutz besonders wichtig. Beachten Sie folgende Grundsätze: Lors de l'utilisation de bots IA, la protection des données est particulièrement importante. Respectez les principes suivants :

  • Keine personenbezogenen Daten eingeben: Geben Sie niemals Namen, Adressen, Noten oder andere persönliche Daten von Lernenden in den KI-Bot ein. Ne pas entrer de données personnelles : N'entrez jamais de noms, adresses, notes ou autres données personnelles d'apprenants dans le bot IA.
  • Anonymisieren: Falls Sie Beispiele brauchen, verwenden Sie fiktive Namen und Daten. Anonymiser : Si vous avez besoin d'exemples, utilisez des noms et données fictifs.
  • Schulkontodaten schützen: Teilen Sie keine Zugangsdaten, Passwörter oder interne Schulinformationen mit dem KI-Bot. Protéger les données du compte scolaire : Ne partagez pas d'identifiants, mots de passe ou informations scolaires internes avec le bot IA.
  • Datenverarbeitung beachten: KI-Bots verarbeiten Ihre Eingaben auf externen Servern. Prüfen Sie die Datenschutzrichtlinien des Anbieters. Tenir compte du traitement des données : Les bots IA traitent vos entrées sur des serveurs externes. Vérifiez la politique de confidentialité du fournisseur.

  • Quellen angeben: Wenn Sie Inhalte aus externen Quellen verwenden, geben Sie die Quelle in Ihrer HTML-Seite an. Citer les sources : Si vous utilisez des contenus provenant de sources externes, indiquez la source dans votre page HTML.

💡 Praktische Tipps 💡 Conseils pratiques

  • Vor der Veröffentlichung: Prüfen Sie den generierten Inhalt auf Korrektheit und Angemessenheit. Avant la publication : Vérifiez l'exactitude et la pertinence du contenu généré.
  • Inhalte lieber selbst formulieren und dem KI-Bot nur die Struktur/Interaktivität überlassen. Préférez formuler les contenus vous-même et ne confier au bot IA que la structure/interactivité.
  • Bei Unsicherheiten bezüglich Urheberrecht: Schulleitung oder Rechtsberatung konsultieren. En cas de doute concernant le droit d'auteur : consultez la direction de l'école ou un conseiller juridique.
  • Creative-Commons-Lizenzen (CC) nutzen und verstehen — diese erlauben oft die freie Nutzung unter bestimmten Bedingungen. Utiliser et comprendre les licences Creative Commons (CC) — celles-ci permettent souvent une utilisation libre sous certaines conditions.
